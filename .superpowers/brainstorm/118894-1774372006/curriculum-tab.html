Design decisions
✓ Tabs: Home · Curriculum · Map — no Progress tab (stats live on Home card)
✓ Active state: filled icon + blue text + soft bg pill
✓ Curriculum navigation: L1 tap → accordion opens subcategories
✓ Subcategory tap → push nav to concept list
✓ "Map ↗" button: tab-switches to Map, scrolls + pulses target node
✓ Progress bar: 2px colored stripe under each zone header, non-pressuring
✓ Endowed progress: "8 / 18" shown — could also be "8 explored" style
