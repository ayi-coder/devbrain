Command Anatomy — two layout options

The middle mode between reference card and type-it challenge. Which layout communicates the structure better?

A

Bracket Labels (top/bottom)

Each part gets a bracket above it with a label below. Like a grammar diagram.

B

Tap-to-Reveal (interactive)

Command is shown fully coloured. User taps each segment to reveal its description — one at a time.

git commit -m "fix login bug"
FLAG
-m
Short for --message. Tells git you're about to give it a description of your changes. Always followed by your message in quotes.
