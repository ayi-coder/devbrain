Design decisions & interaction notes
4-tab navigation

Tab bar expands to Home · Curriculum · Map · Quiz. Lightning bolt icon for Quiz (filled when active). Home hero card retains "N due today" count but tapping it navigates to the Quiz tab — it no longer launches directly.

Session builder (max 5)

User taps concept cards to add them to the 5-slot session builder. Slots show as dashed "+" placeholders when empty. Chips appear with a × to remove. Start Quiz button is disabled until ≥1 concept selected. At 5/5, remaining cards dim with "session full" state.

Two recommendation sections

Recommended today — SRS engine output, concepts due or new, shown as cards with zone color bar.
Due for review — overdue concepts sorted by staleness, shown as compact rows with red/orange/yellow day badges.

Concept mini-bar during quiz

A horizontal scroll strip below the progress bar shows all concepts in the session as pills: done (dim), current (highlighted in zone color), upcoming (faded). Gives a sense of position without showing exact question counts.

Map tab — quiz removed

The bottom sheet when tapping a concept node now shows concept info + "Add to Quiz session →" which deep-links to the Quiz tab with that concept pre-loaded. The "Start Quiz →" button is removed from the map bottom sheet entirely.

Lesson screen shortcut

The Curriculum lesson screen replaces "Quiz this concept →" with "Add to Quiz session →" — this deep-links to the Quiz tab and pre-loads the concept into the session builder slot. User still has to tap Start — they keep agency over their session.

Empty state (new users)

If user has zero discovered concepts, the entire recommendations area shows: "Explore the map first — concepts you discover will appear here." The session builder still renders so the layout doesn't jump when concepts are added.

Bridge node deduplication

Bridge nodes (e.g. stdin) appear once in the SRS list regardless of how many zones they belong to. Adding one to a session adds it once. The SRS score is updated once on completion.
